# Havarijný plán
Tento dokument slúži ako havarijný plán pre rozšírenie Lokálny katalóg otvorených dát pre redakčný systém WordPress. Popisovaný produkt nie je možné prevádzkovať mimo prostredia redakčného systému WordPress. Z tohto dôvodu je nevyhnutné aplikovať odporúčané prevádzkové postupy vrátane vhodnej metódy zálohovania a obnovy zo zálohy.

Údaje lokálneho katalógu a jeho nastavenia sú súčasťou primárnych schém poskytovaných redakčným systémom WordPress. Plugin pre lokálny katalóg otvorených dát nevytvára údaje v inej databáze, v súboroch, ani v iných zdrojoch, ktorých by si zálohovanie a obnova žiadala samostatný zreteľ.

Vo všeobecnosti odporúčame vykonávať pravidelnú zálohu všetkých súborov v prevádzkovanom umiestnení a rovnako celú databázu projektu.  Z vytvorených záloh je vhodné vykonať test obnovy minimálne raz a po každej zmene prevádzkového prostredia.

Viac informácií je možné získať v dokumentácii redakčného systému WordPress:
https://wordpress.org/documentation/
